CS1102 -- Course Project -- 2025/2026 Semester B

Project Topic:

LLM for Social Equity: Measuring and Reducing Bias in Language Models

Technologies: HTML, CSS, JavaScript

Group Members:

Lam Kwong Hei -- 60171935
Lo Cheuk Kei -- 58526227
Ng Ka long Ronnie -- 56227373

The purpose of our project is to compare gender and race biases across four large language models (LLMs).
Our primary objective is to determine whether such biases emerge when LLMs are used.
In addition, we sought to evaluate whether modifying prompts could mitigate unfair or biased
outcomes when using these models.

Web Development Techniques Demonstrated

The following techniques were applied across our 16 project files

HTML Techniques

  • Semantic Page StructuringUtilized <header>, <main>, <section>, <article>, <nav>, and <footer> for meaningful document hierarchy
  • Complex Table ConstructionBuilt data tables with <thead>, <tbody>, <tfoot>, colspan/rowspan, and <caption>
  • Interactive Form ElementsImplemented <form>, <input type="radio">, and linked <label> tags for quiz interface
  • Custom Data AttributesLeveraged data-filter, data-type, and data-global-index for dynamic DOM targeting
  • Nested Navigation MenusConstructed multi-level dropdown menus using nested unordered lists
  • Inline Error HandlingApplied onerror attribute on <img> tags for fallback behavior

CSS Techniques

  • Flexbox for AlignmentUsed display: flex with align-items, justify-content, and gap for responsive layouts
  • CSS Grid for GalleriesImplemented grid-template-columns: repeat(auto-fill, minmax()) for fluid image grids
  • Responsive Media QueriesApplied @media breakpoints at 768px and 480px for mobile optimization
  • Advanced Pseudo-ClassesUtilized :hover, :active, :disabled, :nth-child(even), and :first-child
  • Smooth TransitionsCreated hover animations with transition and transform: translateY()
  • Fixed PositioningUsed position: fixed for lightbox overlay and absolute for modal controls
  • Visual EnhancementsApplied linear-gradient(), border-radius, and rgba() shadows

JavaScript Techniques

  • Dynamic DOM ManipulationUsed getElementById, querySelector, createElement, and innerHTML
  • Event Listener ManagementAttached handlers with addEventListener and inline onclick triggers
  • State ManagementTracked application state with global variables (currentPage, currentFilter, score)
  • Advanced Array MethodsProcessed data with .forEach(), .filter(), and .slice() for pagination
  • Template LiteralsGenerated dynamic HTML using ES6 template literals and interpolation
  • Keyboard Event HandlingEnabled lightbox navigation with Escape, ArrowLeft, and ArrowRight keys
  • Form Default PreventionUsed event.preventDefault() for single-page quiz experience
  • Algorithmic PaginationCalculated bounds with Math.ceil(), Math.max(), and Math.min()
